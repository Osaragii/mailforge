# MailForge ✉️

An AI-powered cold email generator built with Next.js, Tailwind CSS, and Groq.

## Features

- 🤖 AI-generated cold emails via Groq (free)
- ⚡ Word-by-word streaming animation
- 🎨 5 tone options (Professional, Friendly, Direct, Humble, Story-driven)
- 📋 One-click copy to clipboard
- 📊 Word count + reading time stats
- 🔄 Regenerate with one click

## Tech Stack

- Next.js 16
- Tailwind CSS v4
- Groq API (llama-3.3-70b-versatile)
- TypeScript

## Getting Started

1. Clone the repo
2. Install dependencies
```bash
   npm install
```
3. Get a free Groq API key at [console.groq.com](https://console.groq.com)
4. Run the dev server
```bash
   npm run dev
```
5. Open [http://localhost:3000](http://localhost:3000)
6. Paste your Groq API key and start generating!
